# Assets

This folder contains demo screenshots and example generated comics for the Comic Panel Generator project.

## Structure

### `demos/`
Screenshots and demos of the application UI:
- Interface screenshots (light/dark mode)
- Feature demonstrations
- User flow examples

### `examples/`
Sample generated comics:
- Sketch outputs (panel layouts)
- Final generated comic images
- Multi-page comic examples

## Guidelines

- **demos/**: Keep UI screenshots up-to-date with the latest interface
- **examples/**: Include diverse examples showing different styles and use cases
- Use descriptive filenames (e.g., `dark-mode-interface.png`, `action-comic-example.png`)
- Optimize images for web (reasonable file sizes)
